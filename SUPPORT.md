# Support

## Before opening an issue

1. Check `docs/guides/` and the README quick start.
2. Confirm your OS is Windows 11 x64 for MVP builds.
3. Redact secrets from logs before attaching them (see `docs/guides/share-diagnostics.md`).

## Channels

| Need | Channel |
|------|---------|
| Bugs / features | GitHub Issues (once public) |
| Security | SECURITY.md private process |
| Discussions | GitHub Discussions (once public) |

Private beta: contact the repository owner directly.

## What we do not support in MVP

- macOS / Linux builds
- Scraping authenticated web UIs
- Automatic full-repo upload to any provider
